# ShopFlow — E-Commerce Microservices

## Tech Stack
Java 17 | SpringBoot 3 | MySQL | Docker | AWS

## Services
- User Service (Port 8081) — JWT Auth
- Product Service (Port 8082) — CRUD + Redis
- Order Service (Port 8083) — Kafka + OpenFeign
- Payment Service (Port 8085) — Mock payments

## Features
- JWT Authentication
- BCrypt password encryption
- Inter-service communication (OpenFeign)
- RESTful APIs
